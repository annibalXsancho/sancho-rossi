Sur votre téléphone / un autre appareil

1 — Les deux appareils doivent être sur le même Wi-Fi.
2 — Sur le Mac, le serveur doit être lancé en écoute réseau : python3 -m http.server 4173 --bind 0.0.0.0 depuis le dossier de l'app.
3 — Sur le téléphone, ouvrez http://ADRESSE-IP-DU-MAC:4173 (l'adresse s'obtient avec ipconfig getifaddr en0).
4 — Menu du navigateur → « Ajouter à l'écran d'accueil » : l'app s'installe avec son icône, plein écran, comme une application native.
⚠ En accès réseau non sécurisé (http), le navigateur du téléphone bloque la géolocalisation et le wake lock. Pour la navigation GPS sur le terrain, hébergez l'app en HTTPS (Netlify / GitHub Pages, gratuit — le dossier se dépose tel quel) : toutes les fonctions sont alors disponibles hors ligne après la première visite.

Application

Sancho Rossi v4 — PWA installable. Itinéraires balisés : géométrie réelle OpenStreetMap (~15 m). Profils d'altitude : données réelles Open-Meteo. Vue 3D : relief Terrarium + imagerie Esri. Aucune donnée ne quitte cet appareil sans votre action.
